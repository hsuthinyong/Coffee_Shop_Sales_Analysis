# Coffee_Shop_Sales_Analysis
End-to-end sales analytics project featuring SQL data cleaning, KPI calculations, trend analysis, and interactive Excel dashboards for a coffee shop business.

## Tools Used
- MySQL (data querying and cleaning)
- Excel (pivot tables, dashboard, visualizations)

## What I Did
- Queried and cleaned transaction-level sales data
- Built an interactive dashboard with timeline filtering
- Identified key revenue drivers and sales trends

## Files
- `coffee_shop_sales_data_query.sql`
  Contains all SQL queries used for analysis

- `coffee_shop_sales_dashboard.xlsx`
  Excel workbook with charts, pivot tables and dashboard
  
- `dashboard_preview.png`
  Image of dashboard
